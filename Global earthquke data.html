This page displays real-time global earthquake data retrieved from the U.S. Geological Survey (USGS) open API. It includes earthquakes of magnitude 2.5 and above.

  • All data is automatically fetched and updated from the USGS earthquake feed.
  • Updates occur approximately every 5 minutes to reflect recent seismic activity.
  • Each record includes time (UTC), magnitude, location, depth, and epicenter coordinates.
  • Epicenter locations are based on USGS calculations and may slightly differ from local observations.
  • The map uses OpenStreetMap to visualize earthquake events worldwide.

Disclaimer:
EarthWatch is an academic project and not an official disaster warning system. For local emergencies, please refer to official government or meteorological authorities.

Data Sources: USGS Earthquake API, OpenStreetMap
Event updates may experience slight delays due to external data provider response time.
